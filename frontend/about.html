Project Overview

TripGenius is a smart travel planning web application that generates optimized travel itineraries based on user preferences, budget, duration, and number of travelers. The system focuses on intelligent planning rather than real-time bookings.

Inspiration

The project draws inspiration from platforms like MakeMyTrip, TripAdvisor, and Booking.com. Unlike commercial platforms, TripGenius emphasizes logical decision-making, itinerary optimization, and user-centric design.

Key Features

  • Preference-based itinerary generation
  • Budget-aware travel planning
  • Support for Indian and international destinations
  • Day-wise optimized itineraries
  • Simple and responsive UI

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Logic: Rule-based & AI-inspired planning
  • Storage: Browser Local Storage
  • Development Tool: Visual Studio Code

Academic Purpose

This project is developed as part of a final-year academic submission to demonstrate frontend development skills, logical problem-solving, and modular web application design.
